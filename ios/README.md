# iOS Platform Status

This iOS plugin implementation is kept for historical reference. It uses the older Flutter plugin and CocoaPods-era layout from this repository.

Active iOS support for `tflite_flutter`, including Swift Package Manager adoption and future Flutter packaging changes, should be tracked in the TensorFlow repository:

- Package: https://pub.dev/packages/tflite_flutter
- Repository: https://github.com/tensorflow/flutter-tflite
- Issues: https://github.com/tensorflow/flutter-tflite/issues
